# Medical Appointment App

## Objetivo general
Desarrollar una aplicación web para la gestión de citas médicas, permitiendo registrar, organizar y administrar citas entre pacientes y personal médico de forma eficiente, utilizando el framework Laravel y herramientas modernas de desarrollo web.

## Tecnologías a utilizar
- Laravel
- PHP
- Composer
- Jetstream
- Livewire
- MySQL
- HTML5
- CSS3
- JavaScript
- Git y GitHub
- Visual Studio Code
